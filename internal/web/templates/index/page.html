Staircase

Welcome to Staircase

Staircase is a tool for building and managing data pipelines.

Get started by creating a new pipeline or exploring the documentation.

Datasources

A datasource represents a connection to an external postgres database. It contains the necessary information to connect and interact with the database, such as host, port, username, and password. You can create, and delete datasources as needed to manage your data connections effectively.

Jobs

A job is the core of the Staircase system. At it's core it contains the Staircase declaration, a custom domain specific language (DSL), which it uses to navigate a website's DOM and extract data.
